# Horarios App 🕒

Una aplicación web moderna y flexible para la gestión de horarios, actividades y turnos. Diseñada para ser intuitiva, rápida y fácil de compartir.

## ✨ Características

- **Gestión Multi-hoja**: Crea múltiples pestañas para organizar diferentes tipos de horarios (actividades, turnos, deportes, etc.).
- **Plantillas Predefinidas**: Comienza rápidamente con plantillas para:
  - Hoja en blanco
  - Actividades semanales
  - Gestión de turnos
  - Entrenamientos deportivos
- **Privacidad Total (Sin Servidores)**: Tus datos nunca salen de tu navegador. No hay bases de datos externas ni registros; todo se gestiona de forma local.
- **Persistencia en URL**: Tus datos se guardan automáticamente en la URL. ¡Simplemente copia y comparte el enlace para que otros vean tu horario!
- **Generación de Códigos QR**: Comparte tus horarios fácilmente escaneando un código QR generado automáticamente.
- **Modo de Impresión**: Vista optimizada para imprimir tus horarios en papel de forma clara y profesional.
- **Interfaz Moderna**: Construida con Next.js, Tailwind CSS y componentes de shadcn/ui, ofreciendo una experiencia de usuario fluida y adaptativa (responsive).

## 🚀 Tecnologías Utilizadas

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Lenguaje**: [TypeScript](https://www.typescriptlang.org/)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Componentes**: [shadcn/ui](https://ui.shadcn.com/)
- **Iconos**: [Lucide React](https://lucide.dev/)
- **Gestión de Estado en URL**: [lz-string](https://pieroxy.net/blog/pages/lz-string/index.html) para compresión de datos.

## 🛠️ Instalación y Desarrollo

1. Clona el repositorio.
2. Instala las dependencias:
   ```bash
   npm install
   # o
   pnpm install
   # o
   yarn install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
